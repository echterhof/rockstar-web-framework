Welcome

This is a static HTML file served by the Rockstar Web Framework. The framework provides powerful static file serving capabilities with support for multiple protocols and advanced features.

Framework Features

  • Multi-Protocol Support (HTTP/1, HTTP/2, QUIC)
  • Multiple API Styles (REST, GraphQL, gRPC, SOAP)
  • Built-in Caching and Session Management
  • Database Integration with Multiple Drivers
  • Internationalization (i18n) Support
  • WebSocket Support
  • Plugin System for Extensibility
  • Middleware Pipeline
  • Multi-Tenancy Support
  • Comprehensive Error Handling
  • Monitoring and Metrics
  • Static File Serving

Getting Started

To use static file serving in your application, configure the framework to serve files from a directory:

router.Static("/static", pkg.NewOSFileSystem("./public"))
